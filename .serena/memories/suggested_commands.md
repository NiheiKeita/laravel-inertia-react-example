# Suggested Commands

- Shell commands should be prefixed with `rtk` per `/Users/niheikeita/.codex/RTK.md`; use `rtk proxy <cmd>` when RTK filtering blocks a command shape.
- Root Docker shortcuts: `make up`, `make bash`, `make stop`, `make ps`.
- From `src/`: `npm ci`, `npm run dev`, `npm run build`, `npm run lint`, `npm run lint:fix`, `npm run type-check`, `npm run storybook`, `npm run test-storybook`.
- From `src/`: `composer install`, `composer phpcs .`, `composer phpcs-fix`, `vendor/bin/phpstan analyze`, `vendor/bin/phpunit`.
- Prefer `rg` / `rg --files` for source discovery.
